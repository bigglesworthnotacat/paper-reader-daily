## 摘要
零样本物体目标导航（ZS-OGN）要求具身智能体在没有事先训练的情况下探索并定位目标物体。为此，近期方法采用了基础模型。但它们通常依赖静态先验且缺乏适应性，导致重复错误和代价高昂的试错。在本文中，我们提出了一种自演化的ZS-OGN框架，能够实现持续的测试时改进。具体而言，我们通过从过去轨迹中提取可操作知识，构建了一个智能体规则记忆。然后，我们提出了一种基于上置信界的检索策略，通过平衡语义相关性和历史成功率来选择有效规则。此外，我们引入了一个记忆引导的预反思模块，在行动前预测潜在结果，从而减少低效探索。大量实验表明，我们的方法优于现有的零样本基线，成功率提高了10.1%，同时减少了不必要的步骤。

## Abstract
Zero-Shot Object-Goal Navigation (ZS-OGN) requires embodied agents to explore and locate target objects without any prior training. To this end, recent methods leverage foundation models. But they typically rely on static priors and lack adaptation, which leads to repeated errors and costly trial and error. In this paper, we propose a self-evolving ZS-OGN framework that enables continuous test-time improvement. Specifically, we build an agentic rule memory by extracting actionable knowledge from past trajectories. Then, we propose a retrieval strategy based on upper confidence bound, selecting effective rules by balancing semantic relevance and historical success. In addition, we introduce a memory-guided preflection module that forecasts potential outcomes before action, reducing inefficient exploration. Extensive experiments show that our method outperforms existing zero-shot baselines, achieving a 10.1\% improvement in success rate with fewer unnecessary steps.

- **问题背景**：零样本物体目标导航（ZS-OGN）要求智能体在无任何预训练的情况下，在未知环境中定位指定类别的目标物体（如“沙发”），是具身人工智能的基础任务。
- **现有方法局限**：
  - 训练型方法依赖大量环境交互和特定微调，泛化能力弱。
  - 训练自由（Zero-Shot）方法借助大型语言/视觉-语言模型（LLMs/VLMs）进行语义规划，但通常依赖固定的静态先验，无法从实时反馈中自适应，容易在类似场景中**重复犯相同错误**。
  - 缺乏动作前的风险预判能力，常采用**事后纠错**，导致高成本的试错和有限步数内的低效探索。
- **核心挑战**：如何在有限的测试时步数预算下，**联合实现在线自适应与行动预判**，减少无效物理探索。

### 2. 方法论：EvolveNav 框架
EvolveNav 的核心思想是将导航过程转化为一个**连续的自改进过程**，包含两个主要组件：

#### 2.1 跨回合自演化规则记忆（Inter-Episode Self-Evolving Agentic Rule Memory）
- **规则提取**：每回合结束后（无论成功或失败），利用LLM分析完整轨迹，识别关键决策步，再基于这些步的局部观测和决策上下文，**蒸馏出可泛化的导航规则**（如“避免被视觉相似物误导”“优先搜索客厅”）。
- **语义驱动的信用分配**：
  - 为衡量每步的贡献，提出基于**语义相似度边际增益**的信用分配。第i步的综合语义相似度 \(S_i\) 为：
    \[
    S_i = \alpha \cdot S_{\text{room}}^{(i)} + (1-\alpha) \cdot S_{\text{obj}}^{(i)}
    \]
  - 初始信用 \(C_i = S_i - S_{i-1}\)，经裁剪和Softmax归一化后，得到各步权重 \(w_i\)。一条新规则 \(r\) 的初始支持分数 \(S_r\) 为关联关键步的加权和。
- **基于UCB的动态记忆管理**：
  - 随着规则记忆库增长，将规则检索建模为**多臂老虎机问题**，使用上置信界（UCB）平衡利用与探索。
  - 规则 \(r\) 的期望效用 \(\mu_r\) 通过动量法更新：\(\mu_r \leftarrow \eta \cdot \mu_r + (1-\eta) \cdot S_r\)。
  - UCB 分数计算：
    \[
    \text{UCB}(r) = \mu_r + \beta \sqrt{\frac{\ln T}{N_r}}
    \]
    其中 \(T\) 为总回合数，\(N_r\) 为规则被检索次数，新规则（\(N_r=0\)）UCB初始化为∞。
  - 每次决策时，检索UCB评分最高的Top-K规则注入LLM上下文。

#### 2.2 记忆引导的预反思探索（Memory-Guided Preflection）
- **行动前风险预测**：在智能体执行动作前，LLM根据当前各候选前沿（Frontier）的语义观测和检索到的Top-K历史规则，**明确预测每个前沿可能导致的失败风险**（如死路、不相关区域）。
- **主动决策**：强制LLM先进行风险评估，再结合探索收益选择最优、低风险的前沿方向，从而**将范式从被动纠正转变为主动风险规避**。
- **系统支撑**：通过RGB-D与里程计增量构建2D占据栅格地图，提取候选前沿；低级控制器基于局部目标点导航，仅在必要时调用LLM，且设有最大执行步数防止卡死。
