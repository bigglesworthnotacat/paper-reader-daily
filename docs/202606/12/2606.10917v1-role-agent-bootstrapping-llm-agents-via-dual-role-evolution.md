## 摘要
尽管大语言模型（LLM）智能体在复杂任务上展现了强大性能，但其学习常因低效的交互反馈和静态训练环境而受限，阻碍了更广泛的泛化。为解决这些限制，本文提出Role-Agent，一个利用单个LLM同时充当智能体与环境的框架，实现自举式协同演化。Role-Agent包含两个协同组件：World-In-Agent（WIA）和Agent-In-World（AIW）。在WIA中，LLM作为智能体，在每次动作后预测未来状态；预测状态与实际状态的对齐程度被用作过程奖励，从而鼓励环境感知推理。在AIW中，LLM分析失败轨迹中的失败模式，并检索具有相似失败模式的任务，进而重塑训练数据分布以进行针对性训练。在多个基准上的实验表明，Role-Agent持续提升性能，相较于强基线平均增益超过4%。

## Abstract
Although Large Language Model (LLM) agents have demonstrated strong performance on complex tasks, their learning is often limited by inefficient interaction feedback and static training environments, which hinder broader generalization. To address these limitations, this paper introduces Role-Agent, \textcolor{black}{a framework} that harnesses a single LLM to function concurrently as both the agent and the environment, enabling a bootstrapped co-evolution. Role-Agent comprises two synergistic components: World-In-Agent (WIA) and Agent-In-World (AIW). In WIA, the LLM acts as the agent and predicts future states after each action; the alignment between predicted and actual states is then used as a process reward, encouraging environment-aware reasoning. In AIW, the LLM analyzes failure modes from failed trajectories and retrieves tasks with similar failure patterns, thereby reshaping the training data distribution for targeted practice. Experiments on multiple benchmarks show that Role-Agent consistently improves performance, yielding an average gain of over 4\% over strong baselines.

### 2. 论文提出的方法论
核心思想是“双角色自举演化”，包含两个协同模块：World-In-Agent (WIA) 和 Agent-In-World (AIW)。

*   **World-In-Agent (WIA) - 世界在智能体**
    *   **核心思想**：让LLM在作为智能体决策时，不仅要产生行动，还要预测该行动导致的环境未来状态，并通过预测准确性来调制奖励信号，从而鼓励智能体进行更具环境感知的推理。
    *   **关键技术细节**：
        1.  **状态预测**：在交互过程中的每一步，智能体在采取行动后，会被提示预测未来 *H* 步的环境状态。
        2.  **预测奖励计算**：通过最长公共子序列（LMS）衡量预测状态与实际状态的对齐程度，生成预测奖励矩阵。
        3.  **奖励调制**：将任务奖励 *R_task* 和预测奖励 *R_pre* 结合，公式为 `R_t = R_task(a_t) * (1 + R_pre(a_t))`。这种乘法调制确保准确的预测能放大正奖励，而错误的预测会衰减奖励，防止仅凭运气获得高回报。
        4.  **状态分组优势**：借鉴GiGPO方法，将轨迹中相同的状态进行分组，计算状态级别的优势函数 `A_S`，并与轨迹级别优势 `A_E` 混合，形成最终的优化优势 `A = A_S + α * A_E`，以更精细地分配功劳。

*   **Agent-In-World (AIW) - 智能体在世界**
    *   **核心思想**：利用同一个LLM充当“环境”角色，分析自身作为智能体时的失败轨迹，识别失败模式，并据此从任务库中检索类似失败模式的任务，动态重塑训练数据分布，实现针对性练习。
    *   **关键技术细节**：
        1.  **失败模式分析**：将失败轨迹和任务描述输入LLM，要求其识别失败的主导类型、触发条件、核心教训，并生成用于检索的查询语句。
        2.  **任务检索与重分布**：将历史上所有失败的任务及其失败模式存储为离线库。对于每次新产生的失败模式，LLM在库中检索出具有相似根本原因（而非表面相似）的任务，并将其重新纳入训练集，增加这些“困难模式”样本的比重。
