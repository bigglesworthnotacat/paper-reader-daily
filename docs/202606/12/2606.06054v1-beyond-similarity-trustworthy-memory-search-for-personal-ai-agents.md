## 摘要
个人AI代理越来越依赖长期记忆来实现跨会话的持续个性化。然而，现有的记忆流程在很大程度上由语义相似性驱动：与当前查询相近的记忆数据被检索并注入模型上下文。这造成了一个关键的可信性缺口，因为语义相关的记忆在上下文上可能仍然不合适，从而导致跨域泄露、谄媚行为、工具调用漂移或记忆诱导的越狱等威胁。本文从信任边界的角度研究了个人AI代理中的记忆搜索。我们评估了代表性的代理记忆框架，包括A-Mem、Mem0和MemOS，以及OpenClaw——一个具有持久状态和工具使用能力的真实个人代理环境。结果表明，长期记忆不仅仅是一个工具层，而是一个持久的控制通道，可以重塑代理解释任务和执行动作的方式，使其极易受到上述威胁的影响。为了缓解这些漏洞，我们提出了MemGate——一个轻量级、可部署的记忆插件，用于可信记忆搜索，仅包含900万个参数，占用35.1MB空间。MemGate被插入到向量记忆存储和主干大语言模型（LLM）之间，无需修改LLM、重写记忆数据库或增加推理时的LLM评判。它对候选记忆表示施加查询条件神经门控，将原始的相似性搜索转变为任务条件记忆准入。在多个主流记忆框架、真实代理设置和多样化的LLM主干中，MemGate减少了记忆诱导的威胁，同时保留了长期记忆的效用。

## Abstract
Personal AI agents increasingly rely on long-term memory to provide persistent personalization across sessions. However, existing memory pipelines are largely driven by semantic similarity: memory data close to the current query is retrieved and injected into the model context. This creates a critical trustworthiness gap, since a semantically related memory may still be contextually inappropriate, leading to threats such as cross-domain leakage, sycophancy, tool-call drift, or memory-induced jailbreaks.   In this paper, we study memory search as a trust boundary in personal AI agents. We evaluate representative agentic memory frameworks, including A-Mem, Mem0, and MemOS, together with OpenClaw, a real-world personal-agent environment with persistent state and tool-use capability. Our results show that long-term memory is not merely a utility layer, but a durable control channel that can reshape how agents interpret tasks and execute actions, leaving them highly susceptible to the aforementioned threats. To mitigate these vulnerabilities, we propose MemGate, a lightweight and deployable memory plug-in for trustworthy memory search, with only 9M parameters and a 35.1MB footprint. MemGate is inserted between the vector memory store and the backbone LLM, requiring no LLM modification, memory-database rewriting, or inference-time LLM judge. It applies a query-conditioned neural gate to candidate memory representations, turning raw similarity search into task-conditioned memory admission. Across multiple mainstream memory frameworks, real-world agent settings, and diverse LLM backbones, MemGate reduces memory-induced threats while preserving long-term memory utility.

## 1. 论文的核心问题与整体含义

- **研究背景**：个人AI代理正从无状态的对话界面演化为具备持久状态的软件助手，其长期记忆是实现跨会话个性化、连续长期辅助的核心基础设施。
- **核心矛盾**：当前主流记忆管道（如A‑Mem、Mem0、MemOS）几乎完全依赖语义相似性（cosine similarity）来检索记忆并注入LLM上下文。这种“相似即准入”的设计忽略了一个关键事实——语义上接近查询的记忆可能在上下文上完全不适合当前任务。
- **问题定义**：相似性不等于可准入性。语义相关但领域错误、隐私暴露、陈旧约束或包含用户主观偏好的记忆，一旦被注入，就会成为隐式的控制信号，扭曲模型推理、语气、安全判断乃至工具调用。记忆搜索不是单纯的召回机制，而是一个 **信任边界**。
- **研究目标**：系统揭示并量化记忆诱导的可信性失败，包括良性过度个性化（跨域泄漏、谄媚、工具调用漂移）和恶意记忆操控（记忆诱导越狱），并提出一种轻量、可部署的防御方法，将无条件的记忆检索转化为任务条件的记忆准入。

## 2. 论文提出的方法论

- **防御位置**：在向量记忆库和主干LLM之间插入一个轻量的**神经门控插件** MemGate，不修改LLM参数、不重写记忆库、不增加推理时的LLM评判调用。
- **核心思想**：对候选记忆的嵌入向量施加**查询条件门控掩码**，有选择地衰减那些与当前任务上下文冲突的维度，从而将原始的语义相似性排序转化为 **任务条件记忆准入**。
- **关键技术细节**：
  - **输入构造**：对于查询嵌入 $\mathbf{q}$ 和记忆嵌入 $\mathbf{v}_m$，构造交互特征 $\mathbf{x}_{\text{inter}} = [\mathbf{q} \oplus \mathbf{v}_m \oplus (\mathbf{q} \odot \mathbf{v}_m)] \in \mathbb{R}^{3d}$。
  - **门控网络**：一个共享MLP（含LayerNorm、SiLU激活、Dropout），将交互特征映射为一个与记忆嵌入同维的连续掩码 $\mathbf{G}_{\phi}(\mathbf{q}, \mathbf{v}_m) \in [0,1]^d$，通过Sigmoid保证各维度在0~1之间。
  - **门控相似度**：利用该掩码计算不对称的门控余弦相似度：
    $$
    S_{\text{MemGate}}(\mathbf{q}, \mathbf{v}_m) = \frac{\mathbf{q} \cdot (\mathbf{v}_m \odot \mathbf{G}_{\phi})}{\|\mathbf{q}\|_2 \|\mathbf{v}_m \odot \mathbf{G}_{\phi}\|_2}
    $$
    其中仅对记忆侧的嵌入进行衰减，查询侧保持不变。
  - **训练目标**：将重排序视为**直接偏好优化（DPO）** 任务。给定查询 $q$、可准入记忆 $m^+$ 和冲突/不安全记忆 $m^-$，使门控策略 $\pi_{\phi}$ 提升对 $m^+$ 的相对检索概率，同时降低对 $m^-$ 的概率。目标函数包含标准DPO损失和对正样本的掩码保持惩罚：
    $$
    \mathcal{L}(\phi) = - \mathbb{E}_D \left[ \log \sigma \left( \beta \log \frac{\pi_{\phi}(m^+|q)}{\pi_{\text{ref}}(m^+|q)} - \beta \log \frac{\pi_{\phi}(m^-|q)}{\pi_{\text{ref}}(m^-|q)} \right) \right] + \lambda \cdot \mathbb{E}_D \left[ \frac{1}{d} \| \mathbf{1}_d - \mathbf{g}^+ \|_1 \right]
    $$
    其中 $\pi_{\text{ref}}$ 为未门控的原生余弦检索策略，$\lambda$ 控制正例掩码偏离全1向量的惩罚。
  - **推理**：对候选记忆重新按门控相似度排序，选取 top‑k 作为最终注入LLM的记忆。
- **理论保证**：证明了正例掩码惩罚能限制有益记忆的相似度漂移（Theorem 1），以及 DPO 优化迫使不安全记忆维度被大幅衰减（Theorem 2）。

## 6. 论文的主要结论与发现
- **记忆诱导威胁普遍且严重**：
  - 良性场景下，引入记忆后跨域泄漏率在多数框架上高达25%以上，谄媚失败率从约19%跃升至44%，工具调用漂移率从约5%飙升至50%以上。

- **MemGate 显著降低威胁且保持效用**：
  - 跨域泄漏率在GPT‑4o‑mini上从27.0%降至3.5%，越狱ASR从16.8%降至4.4%。
  - 同时，LoCoMo整体F1从38.9提升至40.8，个性化基准准确率亦有提升，有益记忆失败率下降。表明 MemGate 并非简单禁用记忆，而是选择性地滤除了有害上下文。
- **效率优越**：MemGate 仅引入微小的检索延迟（OpenClaw 上 P50 延迟从1.47s增至1.59s），且不依赖额外LLM调用。

- **超参数与模块鲁棒性**：适当的 $\lambda$（如0.1）能平衡安全与效用；MemGate 可兼容多种嵌入模型，维度扩展仅带来增量提升。
