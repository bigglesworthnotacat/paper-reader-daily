## 摘要
为大型语言模型（LLM）智能体配备有效技能对于在OpenClaw等现实世界系统中解决复杂任务至关重要。这项工作中，我们旨在开发一个框架，自动构建此类可复用技能，以增强LLM在工具使用、多步推理和动态环境交互方面的能力。为此，我们提出集体技能树搜索（CSTS），这是一种新颖的基于树搜索的技能构建框架，能够构建结构化、多样且可泛化的技能树。CSTS的核心思想是利用集体智能，通过两个迭代阶段联合搜索、识别和组合有效技能：集体技能节点生成（CSN-Gen）和集体技能节点评估（CSN-Assess）。CSN-Gen利用多个模型的集体知识为每个子任务探索多样化的候选技能，实现全面的技能探索。CSN-Assess则采用多个模型作为评判者，通过两种评分机制评估和筛选技能节点：（1）集体质量评分，聚合独立评估以产生对技能效果的稳健估计；（2）集体可迁移性评分，明确验证技能在不同模型间的泛化能力。借助CSTS，我们构建了一套全面的技能树以及技能增强训练数据，使模型能够有效学习和利用技能。此外，我们引入集体技能强化学习，该方法从技能树中主动选择多个相关技能以拓宽解空间的探索，避免因单一技能而陷入同质或次优解。最终，我们训练得到的模型OpenClaw-Skill在长程规划、工具使用和泛化方面展现出卓越的智能体能力，在具有挑战性的基准测试上取得了出色表现。

## Abstract
Equipping Large Language Model (LLM) agents with effective skills is crucial for solving complex tasks in real-world systems like OpenClaw. In this work, we aim to develop a framework that automatically constructs such reusable skills to enhance LLMs in tool use, multi-step reasoning, and dynamic environment interaction. To this end, we propose Collective Skill Tree Search (CSTS), a novel tree-search-based skill construction framework that constructs structured, diverse and generalizable tree of skills. The core idea of CSTS is to leverage collective intelligence to jointly search, identify and compose effective skills via two iterative phases: Collective Skill Node Generation (CSN-Gen) and Collective Skill Node Assessment (CSN-Assess). CSN-Gen exploits collective knowledge from multiple models to explore diverse candidate skills for each subtask, enabling comprehensive skill exploration. CSN-Assess employs multiple models as judges to evaluate and select skill nodes with two scoring mechanisms: (1) collective quality scoring that aggregates independent evaluations to produce a robust estimate of skill effectiveness, and (2) collective transferability scoring that explicitly verifies whether a skill generalizes well across different models. With CSTS, we construct a set of comprehensive tree of skills along with skill-augmented training data, enabling models to effectively learn and utilize skills. Besides, we introduce Collective Skill Reinforcement Learning, which actively selects multiple relevant skills from the tree to broaden solution-space exploration, avoid being trapped by a single skill and its resulting homogeneous or suboptimal solutions. As a result, our trained model, OpenClaw-Skill, exhibits outstanding agentic capabilities in long-horizon planning, tool use and generalization over challenging benchmarks.

- **研究背景与动机**  
  大型语言模型（LLM）在构建能够执行工具调用、多步推理与动态环境交互的智能体时，面临一个关键挑战：如何自动获得**可复用的结构化技能**。现有的技能构造方法（如 SkillRL、Trace2Skill、CoEvoSkills）存在三个显著局限：  
  - **技能碎片化**——只捕获孤立子任务的局部过程，缺乏对长程依赖的显式编排；  
  - **技能多样性受限**——技能往往从单一模型的少量轨迹中提炼，隐含该模型的偏好偏差；  
  - **技能可迁移性差**——当切换到底层不同的 LLM 骨干时，已有技能的性能明显下降。  
  针对这些问题，本文以 OpenClaw 等真实交互系统为场景，提出一个**自动构建结构化、多样且可泛化的技能树**的框架，旨在系统性提升 LLM 智能体在长程规划、工具使用及任务泛化上的能力。

## 2. 方法论设计与关键技术细节
- **总体框架：Collective Skill Tree Search (CSTS)**  
  将复杂任务分解为有序子任务序列 \(T \rightarrow (t_1, t_2, \dots, t_M)\)，然后在每个子任务层上迭代执行两个阶段：**集体技能节点生成（CSN-Gen）** 与**集体技能节点评估（CSN-Assess）**，最终构建一棵“技能树”：树的一层对应一个子任务，每个节点是一个候选技能，一条从根到叶的路径即为一个完整的组合技能方案。

- **CSN-Gen：集体技能节点生成**  
  给定一组模型 \(\mathcal{M} = \{M_1, M_2, \dots, M_N\}\)，每个模型 \(M_n\) 尝试求解同一子任务 \(t_m\)，产生执行轨迹 \(\tau_{m,n}\)（包含推理状态、动作、观察和最终奖励）。利用一个共享的技能合成器 \(\Phi_{skill}\) 将每条轨迹抽象为一个候选技能节点 \(s_{m,n}\)，描述适用上下文、必需输入、推荐动作、预期输出、验证标准和恢复策略。该过程并行执行以提升效率并保证候选技能的多样性。

- **CSN-Assess：集体技能节点评估**  
  从两个维度对候选技能评分并筛选：
  1. **集体质量评分**：多个评判模型独立为技能的可理解性、可执行性、完整性和相关性打分，聚合为 \(Q_{m,n} = \frac{1}{J}\sum_{j=1}^{J} q^j_{m,n}\)；
  2. **集体可迁移性评分**：将模型 \(M_n\) 生成的技能 \(s_{m,n}\) 提供给其他模型 \(M_k(k \neq n)\)，要求它们在附加该技能的条件下重新执行同一子任务，记录验证分数 \(r^{m,n}_k\)，取平均 \(Tran_{m,n} = \frac{1}{N-1}\sum_{k \neq n} r^{m,n}_k\)。
  最终节点得分 \(Score(s_{m,n}) = Q_{m,n} + Tran_{m,n}\)，选择得分最高的节点 \(s^*_m\)，组成任务 \(T\) 的组合技能路径 \(S^*_T = (s^*_1, s^*_2, \dots, s^*_M)\)。

- **基于技能路径的监督微调（SFT）**  
  利用选中技能路径和对应的示范轨迹构建 SFT 数据，训练模型初步掌握结构化过程知识：  
  \[
  \mathcal{L}_{SFT}(\theta) = -\mathbb{E}_{(T,S^*_T,\tau^*_T)\sim D_{SFT}} \log \pi_\theta (\tau^*_T \mid T, S^*_T)
  \]

- **集体技能强化学习（CSRL）**  
  在 SFT 基础上进一步提升模型从多个候选技能中挑选最优策略的能力。对每个子任务 \(t_m\)，从技能集中选取若干技能 \(\{s_{m,n}\}\)，使用旧策略为每个技能生成 \(G\) 条 rollout，形成集体技能条件组 \(B_m\)。计算 rollout 的奖励后，在**整个组内**进行相对优势归一化：
  \[
  A^{m,n}_g = \frac{r^{m,n}_g - \mu_m}{\sigma_m + \delta}
  \]
  然后用 GRPO 风格的目标函数优化策略，使得具有正向跨技能优势的动作概率被提高，从而鼓励模型采用更有效的技能条件策略。最终训练得到的模型称为 **OpenClaw-Skill**。
