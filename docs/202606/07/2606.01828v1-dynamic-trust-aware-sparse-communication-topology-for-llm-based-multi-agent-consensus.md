## 摘要
大型语言模型驱动的多智能体系统通过多轮审议、角色专门化和交叉验证，增强了复杂推理任务的可靠性。然而，现有的多智能体辩论和协作框架通常采用全连接通信，导致消息数量、令牌成本和端到端延迟随智能体数量近似二次增长；虽然固定稀疏拓扑减少了开销，但它们无法使通信关系适应不同的任务实例或中间推理状态，从而容易保留低价值交互或丢失关键的纠错信息。为解决这一问题，本文提出了DySCo（动态稀疏共识），一种动态信任感知的稀疏共识机制。在每一轮推理中，DySCo基于智能体可靠性、答案分歧度和任务相关性估计通信边的价值，并在预算约束下选择少量高价值边进行消息交换；然后通过动态信任权重聚合不同智能体的答案，一旦共识稳定即提前终止讨论。该机制以按需通信替代了全局广播，从而在保留关键交叉验证信息的同时降低了通信开销。我们进一步给出了通信复杂度和共识稳定性的分析，并在数学推理、逻辑推理和事实问答任务上评估了DySCo的性能。

## Abstract
Large language model-driven multi-agent systems enhance the reliability of complex reasoning tasks through multi-round deliberation, role specialization, and cross-validation. However, existing multi-agent debate and collaboration frameworks typically adopt fully connected communication, causing the number of messages, token costs, and end-to-end latency to grow approximately quadratically with the number of agents; although fixed sparse topologies reduce overhead, they cannot adapt communication relationships to different task instances or intermediate reasoning states, making them prone either to preserving low-value interactions or to losing critical error-correction information. To address this problem, this paper proposes DySCo (Dynamic Sparse Consensus), a dynamic trust-aware sparse consensus mechanism. In each round of reasoning, DySCo estimates the value of communication edges based on agent reliability, answer divergence, and task relevance, and selects a small number of high-value edges for message exchange under budget constraints; it then aggregates the answers of different agents through dynamic trust weights and terminates the discussion early once consensus stabilizes. This mechanism replaces universal broadcasting with on-demand communication, thereby reducing communication overhead while preserving essential cross-validation information. We further present analyses of communication complexity and consensus stability, and evaluate the performance of DySCo on mathematical reasoning, logical reasoning, and factual question-answering tasks.

## 1. 论文核心问题与整体含义

大型语言模型驱动的多智能体系统（如多智能体辩论 MAD、ChatDev、AutoGen 等）通过角色分工、多轮交互和交叉验证显著提升了复杂推理的可靠性。然而，当前主流框架普遍采用全连接通信（每个智能体与所有其他智能体交换消息），导致以下瓶颈：

- **通信代价爆炸**：消息数量、令牌开销和端到端延迟随智能体数量 n 近似二次增长（O(Rn²)），难以规模化。
- **固定拓扑的僵化**：现有工作尝试用固定稀疏拓扑（环、星型、随机图）降低开销，但通信关系无法根据**任务实例**和**中间推理状态**动态调整——要么保留大量低价值交互，要么丢失关键的纠错信息。

因此，研究核心问题为：**如何在通信预算约束下，根据智能体当前状态动态选择高价值的通信边，实现高效而可靠的共识收敛？** 论文提出的 DySCo 机制正是通过“动态信任感知稀疏通信”来同时优化推理质量与通信成本，从而在精度与效率间取得更优平衡。

## 2. 方法论

DySCo 整体上由四个模块组成，对应一轮推理中的**独立初始化→动态边选择→信任感知更新→共识聚合与早停**。

### 2.1 智能体状态建模

每个智能体 a_i 在第 t 轮维护三元组 z_i^t = (y_i^t, r_i^t, c_i^t)，分别表示当前答案、自然语言推理链和自报/校正后的置信度（[0,1]）。

### 2.2 独立初始化

第 0 轮各智能体独立求解任务 x（统一提示 P_solve），产生初始答案、推理和置信度。智能体可以是同一 LLM 的不同采样实例，或通过不同系统提示、角色赋予的异构智能体。

### 2.3 动态通信边选择（核心创新）

对每条有向边 j→i，计算边价值得分 s_{ij}^t：

\[
s_{ij}^t = \alpha T_j^t + \beta c_j^t + \gamma D_{ij}^t + \delta H_{ij}(x) - \eta L_j^t
\]

- \(T_j^t\)：发送方 j 的历史信任权重；
- \(c_j^t\)：j 当前置信度；
- \(D_{ij}^t\)：i 与 j 之间的答案/推理分歧度（如多选题指示函数、推理嵌入余弦距离）；
- \(H_{ij}(x)\)：任务依赖/角色互补度（例如代码审查中测试者对编码者的依赖）；
- \(L_j^t\)：预期接收 j 消息的令牌成本；
- \(\alpha, \beta, \gamma, \delta, \eta\) 为可调超参数。

接收方 i 选择得分最高的 k 个发送方，构成邻居集合 \(N_i^t = \text{TopK}_{j\neq i}(s_{ij}^t)\)，从而将每轮有向边数控制在 n·k 以内。

### 2.4 压缩批判消息生成

发送方不传递完整推理，而是通过提示 P_critique 生成结构化压缩消息 m_{j→i}^t，包含：当前答案、最关键支持理由、对接收方的一个潜在错误或反例建议（限120词）。这进一步控制了令牌开销。

### 2.5 信任感知状态更新

接收方 i 根据本地状态和邻居消息，结合提示 P_revise 更新答案。输入中包含邻居的信任权重，要求智能体区分“保留原答案”、“部分修改”和“完全改变”三种情况，并对高信任与低信任反馈给予不同考量（低信任仅视为待验证的反例）。

### 2.6 信任权重动态更新

\[
T_i^{t+1} = \lambda T_i^t + (1-\lambda)(\rho V_i^t + (1-\rho) A_i^t)
\]

- \(V_i^t\)：轻量验证器对 i 当前推理的评分（可来自可执行验证器如数学答案检查、代码测试，或独立判断 LLM）；
- \(A_i^t\)：i 与当前多数的临时一致性；
- \(\lambda\) 控制历史平滑度。

### 2.7 共识聚合与早停

最终答案通过信任加权投票得出：\(\hat{y} = \arg\max_y \sum_i T_i^R c_i^R \mathbb{I}[y_i^R = y]\)（开放答案先归一化映射到等价组）。
早停条件：加权答案分布的熵低于阈值 ε，或连续两轮共识答案不变且平均置信度增加微弱。

### 2.8 通信复杂度与稳定性

- 全连接：\(C_{\text{dense}} = O(R n^2 \bar{\ell})\)；
- DySCo：\(C_{\text{sparse}} = O(R n k \bar{\ell}_c)\)，因 k≪n 且压缩消息长度 \(\bar{\ell}_c \le \bar{\ell}\)，大幅降低开销。
- 抽象共识分析：若任意连续 B 轮的通信并图强连通且非零权重有下界 μ，且 LLM 非线性扰动有界且零均值，则所有智能体的置信度向量在期望意义上收敛至共同共识邻域。这一命题仅作理论直觉参考，实际效果由实验验证。

## 6. 主要结论与发现

1. **效率-质量双赢**：DySCo 相比 Dense MAD 在 GSM8K、LogiQA、StrategyQA 上**令牌消耗减少约 70%、延迟减半**，同时准确率均有提升（GSM8K 84.3% vs 82.0%）。
2. **动态拓扑优于固定拓扑**：Static Ring、Random-k 虽降低开销，但因缺失高价值纠错信息，准确率提升有限；DySCo 的动态选择能有效匹配可靠且互补的邻居，取得最佳性能。
3. **信任与分歧信号关键**：消融表明移除信任权重和分歧评分会导致准确率明显下降，说明结合历史正确性和意见多样性至关重要。
4. **消息压缩大幅节流**：压缩批判消息可防止令牌消费暴增（消融中令牌增加 82.2%），且对准确率影响微弱。
5. **早停机制收益明显**：去除早停后额外通信轮次增加成本而无精度增益。
6. **可扩展性优势**：当 n 从 4 增至 12，全连接通信成本飙升且精度在达到极限后下滑，而 DySCo 保持线性成本增长，精度持续上升至 86.8%。
7. **快速收敛**：DySCo 第一轮后共识熵大幅降低（0.62 vs Dense MAD 0.88），并触发早停，既抑制噪声传播又节省开销。
8. **鲁棒性**：超参数在合理区间内表现稳定，不依赖极端调参。

## 7. 优点

- **机制创新**：首次将多智能体 LLM 的通信拓扑从固定结构转变为**随推理状态动态决策的变量**，而非简单稀疏化。
- **多维信号融合**：边评分公式综合考虑信任历史、置信度、答案分歧、任务依赖和通信成本，自适应性更强。
- **系统级设计**：包含消息压缩、信任更新、早停等完整组件，形成端到端可用的共识框架，并附带可复现的提示模板。
- **理论与实验结合**：给出通信复杂度分析和抽象共识条件，同时在三类任务上进行丰富实验，增强了说服力。
- **开销透明且精准**：直接使用令牌消耗和端到端延迟衡量实际成本，而非仅理论消息数。
- **可扩展性突出**：用实验证明随着智能体数量增加，方法仍能获得收益而非退化。

## 8. 不足与局限

- **对 LLM 校准的依赖**：边选择和聚合部分依赖智能体自报置信度 c_t^i，而 LLM 的自评估可能不校准，存在放大过度自信错误的风险。文中建议引入外部验证器校准，但实验未系统评估不同校准策略的影响。
- **信任权重的马太效应**：信任聚合可能产生“富者愈富”现象，早期随机正确或表达自信的智能体可能垄断后续通信，以致少数推理路径被边缘化。没有探索探索机制（如 ϵ-贪婪）的影响。
- **任务适用范围受限**：边价值估计依赖可比答案和可察觉的错误（数学、逻辑、可验证问答），对于开放式生成、价值判断或多解任务，分歧得分和信任信号可能失效。DySCo 被定位为通信协调机制而非通用正确性判定器。
- **模型单一性**：所有实验仅基于 gpt-3.5-turbo，未验证在不同模型族、不同规模、开源模型上的泛化表现，结论可能受模型偏见影响。
- **拓扑连通性风险**：过度稀疏可能导致局部意见簇形成，不再满足联合强连通，共识可能不收敛。文中提及需保留跨簇边，但未实验验证不同连接性条件的稳定性边界。
- **测试集规模有限**：每数据集 300 样本，虽然对于多智能体评估常见，但统计效力可能低于更大规模评测基准。
- **外部验证器依赖性**：信任更新中的 V_i^t 理论上可由外部工具或判断 LLM 提供，但实验未详细说明实际如何实现，这可能影响复现性和结果偏差。
