## 摘要
世界模型为长时域LLM智能体提供了一种原则性的前瞻能力：在执行前预测行动后果。然而，不可靠的前瞻可能被忽略、误用，甚至降低下游决策质量。本文提出WorldEvolver——一种自演化世界模型框架，它在保持下游智能体和所有模型参数冻结的同时，修正部署时的上下文。WorldEvolver集成了三个模块：(i) 情景记忆，通过基于检索的模拟利用真实行动转移；(ii) 语义记忆，从预测-观测不匹配中提取持久启发式规则；(iii) 选择性前瞻，在将低置信度预测整合到智能体推理上下文之前进行过滤。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确率，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个主干网络上取得了最高预测准确率，并在下游智能体成功率上领先其他世界模型基线，证明了测试时记忆修正能同时提升预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：大型语言模型（LLM）智能体在处理长时域任务时，常借助世界模型来预测行动后果（“前瞻”），类似模型基强化学习中的环境模型。然而，世界模型的预测可靠性会随着部署环境变化而下降（分布偏移），导致不可靠的前瞻可能被智能体忽略、误用甚至损害决策。
- **核心问题**：如何在**不更新模型参数**（避免高昂的梯度更新和灾难性遗忘）的前提下，让世界模型在部署过程中**自我进化**，持续提升预测准确度，并为下游智能体**选择性提供可靠前瞻**，从而提升长期规划性能。

### 2. 论文提出的方法论

- **核心思想**：WORLDEVOLVER 框架通过**修正外部记忆上下文**（而非模型权重）实现世界模型的在线自适应。记忆分为情景记忆（Episodic Memory）和语义记忆（Semantic Memory），并加入选择性前瞻（Selective Foresight）机制过滤低置信预测。
- **关键技术细节**：
  - **情景记忆（ME）**：存储已执行的真实状态-动作-下一状态三元组。在预测时，根据候选动作（基于 Jaccard 相似度）检索最相似的 kME 条历史转移，作为上下文提供给冻结的世界模型，以利用先前经验进行类比预测。
  - **语义记忆（MS）**：将预测与观测之间的不匹配（经因子化三元组分解后）转化为启发式规则（如“检查物体不会移动它”），每条规则有证据分数，仅保留分数>0的规则。规则通过 LLM 批评器从批量不匹配中提取，并随新证据动态更新。
  - **选择性前瞻**：计算预测输出的几何平均 token 概率作为置信度 qt，仅当 qt ≥ τ 时才将预测暴露给智能体；否则不提供任何前瞻，从而避免不可靠模拟误导决策。
  - **算法流程**（Algorithm 1）：
    1. 采样草稿动作 a(0)t；
    2. 检索情景记忆；
    3. 冻结世界模型生成预测和置信度；
    4. 应用选择性前瞻得到可见信号 Ft；
    5. 智能体根据 (st, Ft) 采样最终动作 at；
    6. 若 at ≠ a(0)t，重新查询世界模型获得对齐的预测；
    7. 执行动作，更新情景记忆和语义记忆。
