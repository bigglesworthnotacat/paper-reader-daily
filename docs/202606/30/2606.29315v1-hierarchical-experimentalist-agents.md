## 摘要
大型语言模型（LLMs）越来越多地被用于在现实世界中采取行动并支持人类决策，然而大多数智能体依赖参数化知识、固定的训练后数据、检索或搜索。这种范式在新领域以及无法仅凭先验知识回答的复杂查询中会失效。例如，了解物理定律本身并不能使LLMs在复杂物理系统中回答查询或完成长期任务。为了解决这个问题，我们引入了层次化实验者智能体（HExA），一种通过主动实验进行学习的上下文内自我改进框架。HExA迭代地设计和完善与查询相关的实验，从经验中学习可重复使用的可组合技能库，并整合实验证据来回答查询或采取行动。HExA无需训练，兼容任何黑盒模型，且不需要外部监督、预言机或离线数据。为了评估主动实验，我们引入了Interphyre，一个基于PHYRE 2D程序化物理环境的工具调用基准，智能体通过模拟API提出干预措施并测试假设。实验表明，当前的LLM智能体在这些设置中表现不佳，尤其是在Interphyre最难的水平上。Claude Sonnet 4.6仅达到2%的成功率，而HExA将同一模型提升至77%的成功率。HExA也改进了开放权重模型，并优于ReAct和Reflexion等智能体基线。此外，仅使用从较容易水平学习并在没有主动实验情况下迁移的技能，HExA达到了44%的成功率，展示了其学习技能的复用性和泛化性。总体而言，HExA表明通过主动实验学习可以帮助智能体发现有用知识、获取可复用技能，并在新颖的长期任务中取得高效进展。

## Abstract
Large language models (LLMs) are increasingly used to take actions in the real world and support human decision-making, yet most agents rely on parametric knowledge, fixed post-training data, retrieval, or search. This paradigm breaks down in novel domains and for sophisticated queries that cannot be answered from prior knowledge alone. Knowing the laws of physics, for instance, does not by itself enable LLMs to answer queries or complete long-horizon tasks in a complex physical system. To address this, we introduce Hierarchical Experimentalist Agents (HExA), an in-context self-improvement framework to learn from active experimentation. HExA iteratively designs and refines query-relevant experiments, learns a reusable library of composable skills from experience, and integrates experimental evidence to answer queries or take actions. HExA is training-free, compatible with any black-box model, and does not require external supervision, oracles, or offline data. To evaluate active experimentation, we introduce Interphyre, a tool-calling benchmark built on the PHYRE 2D procedural physics environment, where agents propose interventions and test hypotheses through simulation APIs. Experiments show that current LLM agents struggle in these settings, especially on the hardest levels of Interphyre. Claude Sonnet 4.6 achieves only 2% success, while HExA improves the same model to up to 77% success. HExA also improves open-weight models and outperforms agentic baselines such as ReAct and Reflexion. Moreover, using only skills learned from easier levels and transferred without active experimentation, HExA achieves 44% success, demonstrating the reusability and generalization of its learned skills. Overall, HExA shows that learning through active experimentation can help agents discover useful knowledge, acquire reusable skills, and make efficient progress on novel long-horizon tasks.

- **核心问题**：LLM 智能体如何能够**像科学家一样主动实验**——通过探索、提出可证伪假设、与环境交互收集证据，并从中**学习可复用的技能**，从而高效解决复杂、新颖的任务。
- **意义**：作者指出，单纯的参数化知识或检索不足以应对真实世界中广泛的新颖问题，智能体必须拥有**主动实验**的基础能力。

## 2. 方法论：Hierarchical Experimentalist Agents (HExA)

### 2.1 核心思想

- HExA 是一个**无需训练（training-free）** 的**上下文内循环自我改进框架**，完全基于 LLM 的上下文学习，不更新模型权重。
- 核心思路：通过**迭代设计实验** → **与环境交互** → **从结果中提取技能** → **将技能注入后续尝试** 的闭环，逐步积累关于未知环境的可复用策略。

### 2.2 关键技术细节

- **三层组件**：
  - **Actor（执行者）**：任意工具增强的 LLM，负责根据当前任务和技能上下文提出假设、调用工具（模拟器 API）进行实验，生成轨迹（trajectory）。
  - **Evolver（进化者）**：同样为 LLM（可与 Actor 相同），负责分析一批轨迹（成功与失败），对比高/低奖励轨迹，从中提取**策略技能**（原理、适用条件、避免的错误）并更新**技能库**。
  - **Retriever（检索器）**：根据当前任务从技能库中选取最相关的技能（top-M 技能 + top-N 错误记录），注入 Actor 的系统提示中。

- **技能库（Skill Bank）**：
  - 存储两种知识：**策略技能**（成功策略）和**常见错误记录**（失败模式）。
  - 每个技能带有**奖励分数**（由来源轨迹的奖励计算），用于排序和检索。
  - 技能库有容量上限（M_max 技能，N_max 错误），迫使进化者优先保留最有用知识。

- **轨迹奖励函数**：
  - 综合任务成功/失败和迭代效率，使用离散箱（fast success +1.0，slow success +0.25，exploratory failure -0.5，early exit -0.75）。
  - 不对称惩罚：探索性的失败（迭代多）反而比早早放弃的失败获得更高奖励，因为前者提供更丰富信息。

- **技能更新策略**：
  - 两种初始化：Online（空库开始）、Off2On（先收集一批离线轨迹初始化，再在线更新）。
  - 两种更新方式：Evolution（接收旧库+新轨迹，合并/修订/剪枝）、Iterative Replacement（只用新轨迹重建库）。
  - 最优配置为 **Off2On + Evolution**，配合奖励标签。

- **跨任务技能转移**：
  - 给定源水平（source levels）的技能库和目标水平描述，进化者**无需任何目标水平轨迹**，即可合成一个目标技能库。
  - 转移时仅传输抽象物理原理（如杠杆、动量传递），去除源水平的特定坐标和实体绑定。

## 3. 实验设计

### 3.1 基准与数据集

- **Interphyre**：基于 PHYRE 2D 物理模拟环境构建的主动实验基准，包含 8 个不同难度的程序化生成的物理谜题（如 catapult、pass_the_parcel 等）。
- 每个水平有 50~100 个种子（场景配置），智能体需要在最多 25 次交互中放置一个红球，使环境满足目标条件。
- 智能体可通过工具调用 API 观察场景、执行部分/完全模拟、查看接触日志、使用水平特定分析工具。

### 3.2 对比方法

- **Direct**：零样本、无工具，仅凭场景描述一步给出答案。
- **ReAct**：标准推理-行动-观察循环，无跨种子记忆。
- **Reflexion**：在 ReAct 基础上，失败后加入一次口头反思（K=2 尝试）。
- **GRPO**（权重更新基线）：在 Qwen-2.5-3B 上使用 Group Relative Policy Optimization 微调。
- **HEXA**：本文提出的方法（多种配置变体）。

### 3.3 模型

- 前沿闭源：Claude Sonnet 4.6。
- 开放权重：Qwen-2.5-{3B,7B,14B} Instruct、GPT-OSS-{20B,120B}。

### 3.4 评估指标

- **任务成功准确率**（solve rate %）。
- **平均每种子迭代轮次**（效率）。

## 4. 资源与算力

## 6. 主要结论与发现

- **主动实验和技能创建显著提升性能**：在最难的 catapult 水平，Claude Sonnet 4.6 从 Direct 的 2% 提升到 HExA 的 **67.3%**（最高 77%），Reflexion 仅 21.3%。在 pass_the_parcel 上同样大幅提升（60% vs 24%）。
- **技能库提升效率**：HExA 将每种子平均迭代从 ReAct 的 22.9 降至 **14.4**（减少约 37%）。
- **奖励标签至关重要**：去掉奖励信号后，HExA 准确率从 67.3% 降至 50.7%。
- **进化更新优于迭代重建**：Evolution 策略（结合旧库与新的实验）优于每次重建。
- **技能可迁移**：仅用较容易水平的技能库，零次迁移到未见过的新水平，catapult 上从 8%（ReAct）提升至 **44%**（+36 百分点）。
- **HExA 在小样本下优于 GRPO**：在 50 个独特种子下，HExA 在 Down to Earth 上 24% vs GRPO 20%，在 Two Body Problem 上 14% vs 6%。

## 7. 优点

- **无需训练**：完全上下文内学习，兼容闭源模型，无灾难遗忘风险。
- **无外部监督**：仅依靠环境交互反馈，无需 oracle 或离线专家数据。
- **技能层次化与可迁移**：进化机制自然产生从具体到抽象的技能层次，跨任务泛化能力突出。
- **效率高**：小样本场景下优于梯度强化学习方法，尤其适合数据稀缺的新领域。
- **实验设计全面**：消融覆盖了初始化、更新策略、奖励信号等关键维度，结果客观。

## 8. 不足与局限

- **领域局限**：实验仅在 2D 物理模拟中进行，未验证在更广泛主动实验场景（如科学数据分析、交互式编码）中的有效性。
- **依赖 LLM 推理**：技能提取质量受 Evolver LLM 能力上限约束，若 LLM 本身推理弱，则技能质量可能下降。
- **奖励函数简化**：奖励为离散箱，未验证对更复杂连续奖励域的适用性。
- **未与大规模梯度 RL 比较上限**：与 GRPO 仅在小样本对比，HExA 在大预算下能否匹配或超越权重更新方法尚未明确。
- **计算开销**：虽无需梯度训练，但每轮额外 Evolver 调用增加推理成本；未与 ReAct/Reflexion 精确对比总 token 使用量。
- **实验中 Reflexion 参数仅 K=2 尝试，可能限制其表现**；HExA 的 25 次迭代上限也可能对简单水平造成过度计算。
