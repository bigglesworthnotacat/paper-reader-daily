## 摘要
现实世界中的图像修复因复杂且耦合的退化而仍然具有挑战性。尽管最近的代理式图像修复框架利用大型语言模型进行灵活的工具规划，但它们面临两个关键限制。首先，从搜索方案的角度来看，过度依赖贪婪策略无法平衡探索与利用。其次，现有代理系统未能充分利用信息，表现出情节性遗忘。为应对这些挑战，我们提出**自进化代理式图像修复（SEAR）**，将修复问题形式化为一个序列决策问题。受双过程理论启发，SEAR包括一个直觉执行器和一个深思规划器，分别遵循快速思考的**系统1**和慢速思考的**系统2**原则。深思规划器采用剪枝感知的蒙特卡洛树搜索进行长时推理，利用混合无参考奖励和基于多模态大语言模型（MLLM）的锦标赛来防止指标利用。互补地，直觉执行器利用由退化感知状态指纹索引的自进化情节记忆。该机制将昂贵的搜索轨迹蒸馏为自适应专长，在通过记忆重用逐步摊销冷启动探索成本的同时，克服情节性遗忘。在合成和真实世界基准上的大量实验证明了其强大的感知和量化性能。

## Abstract
Real-world image restoration (IR) remains challenging due to complex and coupled degradations. While recent agentic IR frameworks leverage Large Language Models for flexible tool planning, they face two critical limitations. First, from a search scheme perspective, excessive reliance on greedy strategies fails to balance exploration and exploitation. Second, existing agentic systems underutilize information, exhibiting episodic amnesia. To address these challenges, we propose \textbf{Self-Evolving Agentic Image Restoration (SEAR)}, which formulates restoration as a sequential decision-making problem. Inspired by the dual-process theory, SEAR comprises an Intuitive Executor and a Deliberate Planner, respectively following the fast-thinking \textit{System 1} and slow-thinking \textit{System 2} principles. The Deliberate Planner employs Pruning-Aware Monte Carlo Tree Search for long-horizon reasoning, utilizing a hybrid no-reference reward and a Multimodal Large Language Model (MLLM)-based tournament to prevent metric exploitation. Complementarily, the Intuitive Executor leverages a self-evolving episodic memory indexed by degradation-aware state fingerprints. This mechanism distills expensive search trajectories into adaptive expertise, overcoming episodic amnesia while progressively amortizing cold-start exploration costs through memory reuse. Extensive experiments on synthetic and real-world benchmarks demonstrate its strong perceptual and quantitative performance.

## 2. 方法论：核心思想与关键技术细节
- **核心思想**：基于**双过程理论**（Dual-Process Theory），构建“直觉执行器（System 1，快速）”与“深思规划器（System 2，慢速）”协同框架。System 1利用记忆库进行快速记忆驱动推理，System 2通过长程搜索优化复杂场景，并通过自进化记忆将System 2的成果蒸馏到System 1，逐步从昂贵规划过渡到高效执行。
- **关键技术细节**：
  - **问题形式化**：定义工具库A，轨迹τ = (a0,...,a_{H-1})，每个动作at ∈ A，状态转移It+1 = at(It)。最优轨迹最大化终端混合无参考奖励R(I_H) = λ_h HPSv2(I_H) + Σ_j λ_j dIQA_j(I_H)（包括NIQE、MANIQA、MUSIQ、CLIP-IQA，归一化后加权）。

    - 检索：根据查询指纹fq，在L2阈值内选择候选，按潜力得分C(τ_i) = R_max(τ_i) - α||fq - f_i||_2排名。
    - 可靠性门控：执行检索到的轨迹τ_ret，若奖励≥ζ，直接输出；否则以该轨迹作为热启动送入System 2（P-MCTS）。
  - **深思规划器（System 2）**：
    - 宏观规划：LLM根据退化描述生成有序议程T = [T_1,...,T_L]（如先去噪再超分），剪枝搜索空间。
    - 微观规划：**剪枝感知MCTS（P-MCTS）**，在议程约束下进行选择（UCT公式）、扩展（仅从当前子任务工具集采样）、模拟（用后续子任务工具完成look-ahead获得终端奖励）、反向传播（更新Q、N、V_max）。搜索迭代次数N_total = ω·L，ω=9。

  - **双系统协同与记忆进化**：当P-MCTS发现奖励≥η的轨迹τ*时，根据指纹是否已有来新增或更新记忆。记忆容量超限时，淘汰访问次数最少且峰值奖励低于均值的人口。随着经验积累，更多样本可由System 1快速处理。

## 6. 主要结论与发现
- SEAR在几乎所有基准上达到SOTA感知质量（PSNR、SSIM、LPIPS、MANIQA等），尤其在Group B/C和真实数据集上PSNR提升0.3-0.42 dB。

## 7. 优点
- **创新性设计**：将双过程理论引入图像修复，直觉+规划协同，是首个自进化智能体修复框架。
- **技术亮点**：
  - 剪枝感知MCTS（P-MCTS）利用议程剪枝，降低搜索空间，比Vanilla MCTS更高效（工具调用9.93 vs 8.15）。

  - 最优轨迹峰值奖励（R_max）检索策略比平均奖励更鲁棒。

## 8. 不足与局限
- **依赖预设工具库**：只能处理工具库覆盖的退化类型；对于完全未知的退化（如新型噪点）或缺少对应工具时，规划可能失效。
- **冷启动开销**：初始阶段记忆为空，需进行完整P-MCTS搜索，计算成本较高（尽管后续通过记忆摊销）。
