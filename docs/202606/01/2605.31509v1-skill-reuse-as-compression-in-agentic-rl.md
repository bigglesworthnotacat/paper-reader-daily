## 摘要
通过强化学习训练的大型语言模型代理往往会学到脆弱且任务特定的捷径。我们假设，当成功轨迹在结构上可压缩，即被分解为一小组可重用的抽象模式时，代理的泛化能力更强。为形式化这一观点，我们提出ReuseRL，将代理强化学习建立在最小描述长度原则之上。ReuseRL从成功轨迹中提取共享技能词典，并通过分割代价增强强化学习目标，显式惩罚编码效率低下的特异行为。我们为该压缩惩罚导出了一个PAC-Bayes泛化界。在ALFWorld、TextWorld-Cooking和Countdown-Stepwise任务中，ReuseRL相比普通GRPO和强回合长度基线，在分布内和分布外成功率上均有提升。

## Abstract
Large language model agents trained with reinforcement learning (RL) often learn brittle, task-specific shortcuts. We hypothesize that agents generalize better when their successful trajectories are structurally compressible, decomposed into a small set of reusable abstract patterns. To formalize this, we introduce ReuseRL, which grounds agentic RL in the Minimum Description Length (MDL) principle. ReuseRL extracts a shared skill dictionary from successful trajectories and augments the RL objective with a segmentation cost, explicitly penalizing idiosyncratic behaviors that encode poorly. We prove a PAC-Bayes generalization bound for this compression penalty. Across ALFWorld, TextWorld-Cooking, and Countdown-Stepwise, ReuseRL improves in- and out-of-distribution success over vanilla GRPO and strong round-length baselines.

### 2. 论文提出的方法论

- **核心思想**：将代理强化学习建立在**最小描述长度原则**之上。该原则认为，最佳的模型（或这里的技能词典）是能够用最短的代码共同描述数据和模型本身的。论文将这一思想应用于代理的行为轨迹，鼓励学习那些可以被一个共享技能词典高效压缩的成功策略。

- **关键技术细节：REUSE RL 框架**
  - **技能投影**：定义了一个从原始动作到原子技能（如`探索`、`拿取`、`烹饪`）的规则映射函数 φ，将每条复杂轨迹 τ 转换为一个原子技能序列 `s`。
  - **技能词典与分割代价**：维护一个共享的技能词典 `C`，其中包含原子技能和可重用的技能子序列（短语）。对于给定的成功技能序列 `s`，通过动态规划计算在词典 `C` 下覆盖该序列所需的最少短语数量，称为 `seg(s, C)`。**分割代价**则定义为 `SegCost(s | C) = seg(s, C) / T`。如果一个序列由几个长短语组成，其分割代价就低；如果由大量原子技能拼凑而成，代价就高。
  - **EM式交替优化**：
    - **E步（字典提取）**：固定策略 πθ，从当前批次的成功轨迹中，通过最小化一个两部分的描述长度目标 `L_BDL(C)` 来提取最优共享词典 `C`。该目标权衡了词典本身的存储代价和用该词典编码数据所需的代价。实践中，使用贪婪的字节对编码算法来近似求解。
    - **M步（策略更新）**：固定词典 `C`，将分割代价作为额外的奖励信号，修改原始的GRPO目标。新的目标变为最大化 `R(τ) - λ * 1_{R(τ)=1} * SegCost(φ(τ)|C)`，即仅对成功的轨迹施加压缩惩罚，引导策略产生既成功又高度可压缩的行为。

    - **命题1**：证明传统的纯回合长度惩罚是该方法在词典为纯原子技能这一特例下的退化形式。

### 6. 论文的主要结论与发现

- **结构压缩促进泛化**：REUSE RL在ALFWorld（尤其在分布外子集）、TextWorld-Cooking（避免可预见的失败陷阱）和Countdown-Stepwise（形成系统的解题模板）上，均显著优于普通GRPO和强回合长度基线，验证了核心假设。
- **压缩不等于纯粹简短**：纯回合长度惩罚在某些情况下（如TextWorld-Cooking）会适得其反，导致代理陷入脆弱的、导致失败的捷径（如盲目烧焦已经熟的食材）。这证明了有效的压缩必须捕获可重用的结构，而非简单地减少步数。
- **方法有效性**：REUSE RL有效地学习了反映环境特定逻辑的可重用技能短语（如家务中的“拿取-运输-交付”链，烹饪中的“切-炒”步骤），这些习得的结构与现有技能巩固方法（如SkillRL）提炼出的技能高度一致，但REUSE RL是完全内生于RL训练信号的。
- **理论保障**：推导出的PAC-Bayes界为压缩惩罚作为泛化提升的正则化器提供了正式的理论依据。

### 7. 优点

- **新颖且原则性的视角**：将泛化、技能发现和数据压缩统一在最小描述长度原则的框架下，为代理强化学习中的“推理崩溃”问题提供了一个深刻的解释和优雅的解决方案。
- **方法干净、自成一体**：REUSE RL是一个最小化的、即插即用的奖励塑形模块，不需要依赖额外的教师模型蒸馏、外部记忆或复杂的模块，严格地分离并验证了MDL原则的独立贡献。
- **理论与实证结合紧密**：不仅有PAC-Bayes泛化界的理论支持，而且通过丰富的实验和案例分析，从多个维度验证了理论的有效性，揭示了纯长度惩罚失败的根本原因。
- **实验设计扎实**：在多个不同类型和挑战（部分可观察、不可逆动作）的环境上进行了评估，并通过消融实验和深入的案例研究阐明了方法的工作机理。

### 8. 不足与局限

- **技能投影依赖手工设计**：原子技能的提取依赖于为每个环境手工编写的规则映射，这在一定程度上限制了方法的自动化程度和向全新领域的迁移便利性。
- **BPE搜索为近似解**：用于提取字典的贪婪BPE算法是NP-hard问题的近似解。虽然在当前原子技能集大小下效果很好，但在动作空间更庞大的领域，近似误差可能会增大。
- **理论假设限制**：PAC-Bayes界等理论分析依赖于“成功轨迹存在可重用结构”的前提假设。在非结构化或高度新奇的任务中，如果成功轨迹缺乏共享子结构，该方法的效用可能会退化。
- **实验规模和模型限制**：当前实验仅限于文本类基准环境和参数规模在15-17亿左右的中小模型，其在大规模视觉-语言模型或更大参数模型上的效果有待进一步验证。
