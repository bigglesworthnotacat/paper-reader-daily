## 摘要
自我改进的语言智能体通常在孤立环境中评估：智能体尝试任务，接收反馈，并迭代优化自身行为。然而，智能体越来越多地与同伴共同运行，后者的策略和结果公开可见。这引出了一个尚未充分研究的问题：共享经验何时能带来仅靠自我改进无法实现的提升？我们引入 SAGE（社会化智能体群体演化），这是一个评估框架，比较两种计算匹配的条件：SocialEvo，其中来自五个不同模型家族的智能体共同演化，可访问所有同伴的历史记录；以及 SelfEvo，其中每个智能体获得相同数量的任务尝试，但仅能看到自己的过往记录，这与自我改进智能体研究中的常规做法一致。我们在三个领域实例化 SAGE：开放式机器学习研究、长期经济规划以及策略性多人游戏，并在多轮演化中进行评估。我们发现群体历史并非普遍放大器：最强智能体并未超越其自我演化的上限。然而，在自我改进下陷入瓶颈的智能体，在获得同伴经验后可能实现显著突破。在竞争性场景中，反事实对照揭示出智能体普遍得到改进，而非发展出针对特定对手的策略。在不同形式的共享历史中，过滤后的同伴轨迹和反思性摘要往往优于原始日志，表明社交收益取决于抽象能力而非曝光量。这些发现揭示，同伴历史带来的收益因智能体、领域而异，并取决于从公开轨迹中抽象出可迁移知识的能力。

## Abstract
Self-improving language agents are typically evaluated in isolation: an agent attempts a task, receives feedback, and iteratively refines its own behavior. Yet agents increasingly operate alongside peers whose strategies and outcomes are publicly visible. This raises an under-studied question: when does shared experience produce improvements that self-improvement alone cannot achieve? We introduce SAGE (Social Agent Group Evolution),an evaluation framework that compares two compute-matched conditions: SocialEvo, where agents from five distinct model families co-evolve with access to all peers' histories; and SelfEvo, where each agent receives the same number of task attempts but sees only its own past, which is conventional in self-improving agent studies. We instantiate SAGE in three arenas: open-ended ML research, long-horizon economic planning, and strategic multiplayer play, evaluated across multiple evolutionary rounds. We find that group history is not a universal amplifier: the strongest agent does not exceed its self-evolution ceiling. However, agents that plateau under self-improvement can achieve significant breakthroughs when peer experience is available. In competitive settings, counterfactual controls reveal that agents improve generally rather than developing opponent-specific strategies. Across different forms of shared history, filtered peer traces and reflective summaries often outperform raw logs, indicating that social gains depend on abstraction rather than exposure volume. These findings reveal that peer-history gains are agent-specific, arena-dependent, and contingent on the capacity to abstract transferable knowledge from public traces.

### 2. 方法论：SAGE 框架

- **核心思想**：构建固定智能体种群，在完全相同的总计算预算（每轮总尝试次数相同）下，分别运行两种演化模式：
  - **SocialEvo**：K 个智能体共同演化，每轮每个智能体执行 m 次尝试，并可查看所有同伴的公开历史（轨迹、分数、摘要等）。
  - **SelfEvo**：某个焦点智能体单独演化，每轮获得 Km 次尝试（等于 SocialEvo 中群体的总预算），但只能访问其自身的历史。
- **关键变量与抽象**：

  - 配对差异置信区间（SAGE 效果 \(\Delta_i\)），判断各智能体是否获得显著社交增益。

### 6. 主要结论与发现

- **SocialEvo 非普适增益**：群体历史并未提升最强智能体的性能上限；在 SelfEvo 下已表现优异的智能体，在 SocialEvo 中未见额外优势。
- **瓶颈突破者得利**：在自我改进下趋于停滞的智能体（如 GPT‑5.4 和部分 DrugWars 场景），在获取同伴历史后取得了显著的突破性提升。
- **增益因人而异，且可能有害**：同一公开通道对某些智能体是策略发现源，对另一些却是噪声或误导。Kimi‑K2.5 在 DrugWars 中甚至出现显著负增益。
- **对抗场景中的改进是泛化的，而非针对特定对手**：Splendor 的 TEG 分析和影子测试表明，社交演化带来的竞技力提升主要源于**通用能力强化**，而非学习针对某个对手的专门策略（虽然在个别配对中观察到了针对性，但换模型后优势消失，印证了泛化性）。
- **历史信息表示比信息量更重要**：过滤后的冠军轨迹和仅排行榜信号往往优于完整原始日志；反思摘要可能因过度抽象而丢失可执行细节。社交增益依赖于智能体**抽象、过滤和迁移**公开知识的能力，而非简单增加上下文长度。

### 7. 优点

- **计算量匹配的对照设计**：消除了迭代次数差异造成的混淆，真正分离出“同伴经验”的净效应，实验结论可靠。
- **多领域、多模型异构验证**：覆盖合作与对抗、开放式研究与结构化游戏，系统考察社会化演化的跨场景可迁移性。
- **深入的历史表示消融**：不仅检验“有无”历史，更探究“何种形式”的历史最促进学习，为公开信息通道的设计提供了实践指导。
- **反事实诊断工具**：TEG 与影子测试将智能体改进分解为通用进步 vs. 对手针对性，为竞争多智能体分析提供了精细框架。

### 8. 不足与局限

- **智能体种群局限**：实验仅采用五类当前主流模型的 API 版本，未覆盖开源小模型、强定制化脚手架或更广泛的智能体结构，结论分布可能随底层模型变化。
- **演化轮次与样本量有限**：多数实验仅运行 16 轮，且每轮每智能体仅执行 1 次尝试（m=1），可能无法充分捕捉长期社会化效应，结论受随机波动影响。
- **公开历史长度固定且较短**：受推理上下文窗口与预算限制，实验仅测试有界且较短的共享历史，未探讨极长或大规模异构经验下的学习动态。
- **单一固定任务采样**：MLR‑Bench 仅采样一个任务 ID，可能无法代表开放式研究任务的全局特性。
