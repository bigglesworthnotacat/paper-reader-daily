## 摘要
近期的大语言模型（LLMs）在理解和生成代码方面展现出强大能力，涵盖从竞赛编程到仓库级软件工程。在新兴的智能体系统中，代码不再仅是目标输出。它日益成为智能体推理、行动、环境建模以及基于执行的验证的操作基础。我们通过智能体框架的视角来阐述这一转变，并引入代码作为智能体框架：一种将代码作为智能体基础设施核心的统一视角。为了系统性地研究这一视角，我们围绕三个相互关联的层面组织调研。首先，研究框架接口，其中代码将智能体与推理、行动和环境建模连接起来。其次，审视框架机制：用于长周期执行的规划、记忆和工具使用，以及使框架可靠且自适应的反馈驱动控制和优化。第三，讨论将框架从单智能体系统扩展到多智能体设置，其中共享代码工件支持多智能体协调、审查和验证。在这些层面中，我们总结了代码作为智能体框架的代表性方法和实际应用，涵盖编程助手、图形用户界面/操作系统自动化、具身智能体、科学发现、个性化与推荐、开发运维以及企业工作流。我们进一步概述了框架工程面临的开放挑战，包括超越最终任务成功的评估、不完整反馈下的验证、无回归的框架改进、跨多个智能体的一致共享状态、针对安全关键操作的人工监督，以及向多模态环境的扩展。通过将代码置于智能体AI框架的核心，本调研为可执行、可验证且有状态的AI智能体系统提供了统一的路线图。

## Abstract
Recent large language models (LLMs) have demonstrated strong capabilities in understanding and generating code, from competitive programming to repository-level software engineering. In emerging agentic systems, code is no longer only a target output. It increasingly serves as an operational substrate for agent reasoning, acting, environment modeling, and execution-based verification. We frame this shift through the lens of agent harnesses and introduce code as agent harness: a unified view that centers code as the basis for agent infrastructure. To systematically study this perspective, we organize the survey around three connected layers. First, we study the harness interface, where code connects agents to reasoning, action, and environment modeling. Second, we examine harness mechanisms: planning, memory, and tool use for long-horizon execution, together with feedback-driven control and optimization that make harness reliable and adaptive. Third, we discuss scaling the harness from single-agent systems to multi-agent settings, where shared code artifacts support multi-agent coordination, review, and verification. Across these layers, we summarize representative methods and practical applications of code as agent harness, spanning coding assistants, GUI/OS automation, embodied agents, scientific discovery, personalization and recommendation, DevOps, and enterprise workflows. We further outline open challenges for harness engineering, including evaluation beyond final task success, verification under incomplete feedback, regression-free harness improvement, consistent shared state across multiple agents, human oversight for safety-critical actions, and extensions to multimodal environments. By centering code as the harness of agentic AI, this survey provides a unified roadmap toward executable, verifiable, and stateful AI agent systems.
