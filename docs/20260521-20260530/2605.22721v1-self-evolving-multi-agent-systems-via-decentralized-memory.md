## 摘要
自进化多智能体系统（MAS）已成为大语言模型智能体从经验中持续改进的一条有前景的途径，其基础是持久记忆。然而，现有设计几乎无一例外地采用跨智能体共享的集中式存储库，这带来了通信和协调开销、引发隐私问题，并损害了智能体的多样性。我们提出DecentMem，一个去中心化的记忆框架，其中每个智能体维护自己的双池记忆——一个由整合的过去轨迹构成的利用池，以及一个由大语言模型为未见上下文生成的候选者构成的探索池。这两个池根据大语言模型作为评判的阶段式反馈在线重新加权。理论上，我们证明了该设计保证了解决方案空间的全局可达性，并实现了$O(\log T)$的累积遗憾值，与随机赌博机下界在常数范围内匹配。在实践中，在三个MAS框架（AutoGen、DyLAN、AgentNet）、三个Qwen3骨干网络（4B/8B/14B）、两个Gemma4骨干网络（E2B/E4B）以及涵盖数学、代码、问答和具身任务的五个基准测试上，DecentMem相比最强的集中式记忆基线平均准确率提升高达23.8%，相比无记忆基线提升高达52.5%，同时令牌使用量最高减少49%。

## Abstract
Self-evolving multi-agent systems (MAS) have emerged as a promising route to LLM agents that continually improve from experience, with persistent memory at their foundation. However, existing designs almost exclusively adopt a centralized repository shared across agents, incurring communication and coordination overhead, raising privacy concerns, and collapsing agent diversity. We propose DecentMem, a decentralized memory framework in which each agent maintains its own dual-pool memory -- an exploitation pool of consolidated past trajectories and an exploration pool of LLM-generated candidates for unseen contexts. The two pools are reweighted online based on stage-wise feedback from an LLM-as-a-judge. Theoretically, we prove that this design guarantees global reachability of the solution space and achieves $O(\log T)$ cumulative regret, matching the stochastic bandit lower bound up to constants. In practice, across three MAS frameworks (AutoGen, DyLAN, AgentNet), three Qwen3 backbones (4B/8B/14B), two Gemma4 backbones (E2B/E4B) and five benchmarks spanning math, code, QA, and embodied tasks, DecentMem improves average accuracy by up to 23.8% over the strongest centralized memory baseline and by up to 52.5% over the no-memory baseline, while reducing token usage by up to 49%.

### 2. 方法论：核心思想与关键技术细节
DecentMem 的核心是为每个智能体建立一套私有的、不跨智能体共享的记忆生态。

- **双池记忆架构（Dual-Pool Memory）：**
  - **利用池（E-pool）：** 存储该智能体在历史任务中成功整合的轨迹，支持基于相似度检索的直接复用。
  - **探索池（X-pool）：** 一个临时缓冲区，专门由LLM为当前任务生成新颖的候选动作或策略，用于探索从未见过的上下文。
- **在线路由器（Online Router）：**
  - 在每个决策阶段，路由器以概率 $\alpha = w_{E}/(w_{E}+w_{X})$ 选择E-pool，否则选择X-pool。
  - 若选中E-pool，则按语义相似度检索Top-K记忆片段；若相似度不足则自动回退至X-pool。若选中X-pool，则触发LLM生成一条新记忆。
  - 之后智能体基于本地上下文与选出的记忆片段（或新生成记忆）生成具体动作。
- **基于LLM的在线权重更新：**
  - 任务执行完毕后，一个独立的LLM评判者对解决方案进行“分阶段”评分（正确性、分配质量、中间连贯性、最终集成），比较连续两步的得分变化 $\Delta_t = \mathbb{I}[q_{curr} > q_{prev}]$。
  - 若使用E-pool且得分上升，则增加E-pool权重；否则衰减。反之，若使用X-pool且得分上升，则衰减E-pool权重（避免过度依赖过去经验），否则增加。
  - 任务结束后，X-pool中所有记忆被整合进E-pool，X-pool清空，实现经验的持续积累与推送。
- **理论建模：**
  - 将记忆搜索建模为图上带启发式远程传送的随机游走：利用池对应基于相似度的局部漫步，探索池对应LLM先验引发的“传送”。
  - **定理1（全局可达性）：** 在探索池持续激活的前提下，转移矩阵严格正定，保证每个智能体可抵达其局部解空间的任意区域，不会被局部最优永久困住。
  - **定理2（对数级遗憾）：** 将在线路由建模为随机赌博机问题，在环境回报为严格凹函数的假设下，证明DecentMem的路由策略可实现 $O(\log T)$ 的累积遗憾，与赌博机理论下界 $\Omega(\log T)$ 同阶，达到序数最优。
